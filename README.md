AT89S8252
=========

Interrupts:
-----------
interrupt | asm addr. | c
----------|-----------|---
ext0:     | 0003h     | 0
timer0:   | 000Bh     | 1
ext1:     | 0013h     | 2
timer1:   | 001Bh     | 3
serial:   | 0023h     | 4
timer2:   | 002Bh     | 5

Assembler:
----------
```
$NOMOD51
$include (AT898252.h)
org 0000h
LJMP start

; vvvv Interrupt sprungbefehle
org 0003h
LJMP int0_isr

; vvvv Main Program
start:
; ...

; vvvv Interrupt Service Routines
int0_isr:
    ; ...
    RETI

; Programmende
END
```

Ports: `P2.3`  
Konstanten:
`#0A1h` entspr. `0xA1`  
`#1d` entspr. `0d1`
Kommentare: `; Kommentar`

C:
---
vollständige Dokumentation für Cx51 [hier](https://www.keil.com/support/man/docs/c51/c51_intro.htm).
```
#include<AT898252.h>

// data type aliases for convenience, can be ignored
#define u8  unsigned char
#define i8  char
#define u16 unsigned int
#define i16 int
#define u32 unsigned long int
#define i32 long int

// vvvv Function Prototypes, required for Cx51
void init();
void int0();

void main() {
    init();
    // ...
}

// vvvv Setup the interrupts
void init() {
    IT0 = 1; // falling edge triggered
             // state triggered: 0
    EX0 = 1; // Enable EXT0
    EA  = 1; // Enable Interrupts
}

// vvvv Interrupt Service Routine
void int0() interrupt 0 {
    // ...
}
```
